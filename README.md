# About

A collection of Philippine Standard Geographic Code that were broken down by:

- Barangays
- Cities / Municipalities
- Provinces
- Regions

## PSGC Publication Date
- Q4-2021

## Installation
```bash
$ npm install @dctsph/psgc
```
